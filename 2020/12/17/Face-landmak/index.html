Disentangle Representation Learning

1. 什么是DRL

  1. represents data using independent factors of variations in data, which can improve the interpretability and transferability of the representation.

  2. 目标：to discover a set of generative factors, whose elements encoders unique and independent factors of variation in data.

  3. 当前的非监督DRL, 大部分是基于GAN和VAE框架。

    • 主要是用additional regulation to encourage the axes of the latent manifold to align with independent generator factors in the data.
    • 共同特点是：focus on designing the loss function to encourage the factorization of the latent code.

2. 当前存在的挑战

  1. 当前的算法都是在representation learning 和 generation performance之间做Trade-off

    • 因为提高representation Performance的时候会牺牲Generation Performance

    • VAE-based 方法，在Disentangle Representation方面表现非常好，而their generation quality worse than the state-of-the-art.

    • GAN-based 方法，在High-quality synthesis任务上表现的非常好（with a more expressive decoder and without explicit likelihood estimation, 即不需要自监督），然而在Disentangle方面却不尽人意，甚至is difficult to train（目前有很多方法可以to stabilize the training）

3. Background: VQ-VAE

The objective of unsupervised disentangle learning is to describe each data $\color{blue}{x}$ using a set of statistically independent generative factors $\color{blue}{z}$.

  • $\beta$-VAE proposed by optimizing the modified Evidence Lower-Bound (ELBO) of the marginal log-likelihood
    $$\color{blue}{
    E_{x\thicksim p(x)}[\log p(x)] \geq E_{x\thicksim p(x)} [E_{z\thicksim q_\phi(x|z)}[\log p_\theta(x|z)] - \beta D_{KL}(q_\phi(z|x) || p(z) ]
    }$$
